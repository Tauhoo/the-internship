# The intership's homework

## วิธีติดตั้ง

1. ต้องติดตั้ง nod.js ก่อน
2. clone project

```bash
git clone https://github.com/Tauhoo/the-internship.git
```

3. ติดตั้ง dependencies

```bash
yarn #สำหรับ yarn

npm install #สำหรับ ืยท
```

4. ติดตั้ง CLI

```bash
yarn link #สำหรับ yarn

npm link #สำหรับ npm
```
